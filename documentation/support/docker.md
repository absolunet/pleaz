## 1. Usage

> To start Docker service containers, you need to be in the `config/pleaz/` directory

```bash
cd config/pleaz
```

* Start service container:
```bash
docker-compose up -d
```

* Stop/Pause service container:
```bash
docker-compose stop
```

* Clean all service containers:
```bash
docker-compose down
```
